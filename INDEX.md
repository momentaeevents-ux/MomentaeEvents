## 🚀 3-Minute Quick Start

### What You Need:
1. EmailJS account (free)
2. Gmail account
3. 5 minutes of your time

### Steps:
1. **Create Account**: Go to https://www.emailjs.com/ → Sign up
2. **Connect Gmail**: Email Services → Add Gmail → Connect with momentaeevents@gmail.com
3. **Get Key**: Account → API Keys → Copy Public Key
4. **Update Code**: Open js/script.js → Replace placeholder on line 4
5. **Create Template**: Email Templates → Create (copy from TEMPLATE_COPYPASTE.md)
6. **Test**: Fill contact form → Should receive email!
